本项目是内蒙古大学硕士学位论文的 LaTeX 模板 Master-thesis_IMU`的完整实现。模板符合内蒙古大学学位论文撰写规范，支持 XeTeX 与 LuaTeX 引擎，字符编码仅支持 UTF-8。

本示例展示了论文排版中常用的元素，包括公式、表格、算法、参考文献、图片引用等。用户可直接基于此模板撰写硕士学位论文。


-----------------------------------------------------------------
 环境要求

安装 TeX Live 2021 或更高版本（推荐完整安装）
引擎：`XeLaTeX` 或 `LuaLaTeX`


使用方法 

1) 解压 Master-thesis_IMU20260610.rar 将6种字体放入解压后的文件夹
2) 打开主文件Master-thesis_IMU.tex
3) 使用xelatex编译2遍，运行biblatex2遍，再用xelatex编译1遍，保证封面正常显示
